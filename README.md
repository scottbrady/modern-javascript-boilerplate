modern-javascript-boilerplate
=============================

Boilerplate for creating a modern Javascript application.

Included libraries
------------------

This repo includes the following libraries.

* Backbone.js (0.9.9)
* HTML5 Boilerplate (4.0.3)
* jQuery (1.8.3)
* jQuery UI (1.9.2)
* JSON (2)
* Modernizr.js (2.6.2)
* Require.js (2.1.2)
 * text.js
* Underscore.js (1.4.3)
